## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/shreeshpokharel-stack/finance_tracker.git
```

Navigate to the project folder:

```bash
cd finance_tracker
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```

Open in browser:

```text
http://127.0.0.1:5000
```
